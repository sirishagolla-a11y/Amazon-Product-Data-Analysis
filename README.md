# Amazon Product Data Analysis – ApexPlanet Internship Task 2

## Project Overview

This project focuses on Exploratory Data Analysis (EDA), SQL-based business analysis, data visualization, and dashboard design using an Amazon product dataset.

The objective of this project is to uncover patterns, trends, and relationships within the dataset and answer important business questions using SQL.

---

## Dataset Information

The dataset contains Amazon product information such as:

* Product Name
* Product Category
* Product Ratings
* Rating Counts
* Actual Price
* Discounted Price
* Discount Percentage
* Product Reviews

## SQL Business Questions

The following SQL business questions were solved:

1. Top 5 Highest Rated Products
2. Top 5 Products With Highest Discounts
3. Top 5 Most Reviewed Products
4. Average Product Rating by Category
5. Top 5 Most Expensive Products
6. Low Rated Products With High Discounts
7. Number of Products in Each Category

---

## Key Insights

* Electronics categories dominate the dataset.
* Most products have ratings above 4.
* High discounts do not always guarantee high ratings.
* Highly reviewed products tend to have strong customer engagement.
* Premium-priced products mostly belong to electronics categories.
